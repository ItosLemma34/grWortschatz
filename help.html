This project is under development, far from complete. How to use it should be obvious. If in doubt, just click on something.


In its current stage it contains (more-or-less complete) examples from Der Nasse Fisch, Märzgefallene, Lunapark. Examples from Der stumme Tod and Marlow are currently being added. The goal is to include, eventually, all the Gereon Rath Krimis, though that might be too ambitious. (It's a lot of work!) Mal sehen!


This version is posted on Github, under a "strange" name: https://itoslemma34.github.io/grWortschatz/. The point of this bizarre name is to make it hard to find. No search engine looking for anything about Gereon Rath will easily find it. When the program is much further advanced I'll post it under a more informative name. (The current name is unusual for a webside like this one, but not really strange. "itoslemma" is the name of an important mathematical theorem. 34 is a randomly chosen number. "Github" is a very useful online facility for saving and updating programs.)


Please do not give this Github URL to others -- except members of the Chicago Goethe Institute Bookclub.


If anyone reading this is an ace HTML-CSS-JS programmer (I'm just learning!) and would like to help with further development of the structure of the program, please contact me.


Send questions, comments, suggestions, offers to help, errors & bug reports, to me (Ben Crain): bwcrain@gmail.com
