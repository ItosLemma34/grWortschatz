This project is under development, far from complete. If in doubt how to use it, just click on something.


In its current stage it contains examples from Der Nasse Fisch, Der stumme Tod, Märzgefallene, Lunapark. and Marlow. The goal is to include, eventually, all the Gereon Rath Krimis, though that might be too ambitious. (It's a lot of work!) Mal sehen!


This version is posted on Github, under a "strange" name: https://itoslemma34.github.io/grWortschatz/. "itoslemma34" is the name of my Github account. ("Itos Lemma" is an important theorem in stochastic calculus, 34 a random number.) When the program is much further advanced I'll post it under a more informative name, not on Github.
