This project is under development, far from complete. If in doubt how to use it, just click on something.


In its current stage it contains (more-or-less complete) examples from Der Nasse Fisch, Märzgefallene, Lunapark. Examples from Der stumme Tod and Marlow are currently being added. The goal is to include, eventually, all the Gereon Rath Krimis, though that might be too ambitious. (It's a lot of work!) Mal sehen!


This version is posted on Github, under a "strange" name: https://itoslemma34.github.io/grWortchatz/. The point is to make it hard to find. No search engine looking for anything about Gereon Rath will easily find it. When the program is much further advanced I'll post it under a more informative name, not on Github.


Please do not give this URL to others -- except members of the Chicago Goethe Institute Bookclubs.
